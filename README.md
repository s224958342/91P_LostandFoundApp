# SIT708 7.1P Lost and Found App

## Overview
This is an Android Lost and Found mobile application developed for SIT708 Task 7.1P. The app allows users to create, view, filter, and remove lost or found item adverts.

## Features
- Create lost or found adverts
- Save advert details using SQLite database
- Upload and preview item image
- Require image selection before saving an advert
- Select date using DatePicker
- View all adverts in a list
- Filter adverts by category
- Display posted time such as "just now" or "2 min ago"
- View advert details
- Remove advert after the item is returned

## Technologies Used
- Android Studio
- Java
- XML
- SQLite
- ListView
- Spinner
- DatePickerDialog
- Image URI handling

## How to Run
1. Open the project in Android Studio.
2. Sync Gradle files.
3. Run the app on an Android emulator or device.
4. Use the home page buttons to create or view adverts.

## How to Use
1. Open the app.
2. Click **Create a new advert**.
3. Select whether the item is **Lost** or **Found**.
4. Enter the item name, phone number, description, date, location, and category.
5. Click **Upload Image** and select an image.
6. Click **Save** to store the advert.
7. Return to the home page and click **Show all lost and found items**.
8. Use the category filter to filter adverts.
9. Click an advert to view full details.
10. Click **Remove Advert** after the item has been returned.
